# Titanic EDA – Task 3 🚢

## 📌 Mini Exploratory Data Analysis

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to uncover deeper insights into survival patterns.

## 🔍 Work Done

* Data cleaning (handled missing values)
* Feature engineering (Age Groups, Family Size)
* Groupby analysis for insights
* Data visualization using Seaborn & Matplotlib

## 📊 Key Insights

* Survival rate varies across age groups
* Embarkation port affects survival chances
* Medium-sized families had better survival rates
* Strong correlation between Fare, Pclass, and Survival

## 📈 Visualizations

* Age distribution histogram
* Correlation heatmap
* Survival by family size (bar plot)

## 🛠 Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab

Titanic Dataset from Kaggle
